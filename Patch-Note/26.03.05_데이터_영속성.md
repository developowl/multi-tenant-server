## [테넌트 서버 데이터 영속성 강화 업데이트 안내]

안녕하세요, 테넌트 서버의 안정성 향상을 위해 데이터 영속성(Persistence) 유지 업데이트가 **금일 24시**에 진행될 예정입니다.

📌 데이터 저장 위치 안내:
업데이트 이후부터는 접속 시 기본 위치인 홈 디렉토리(/home/tenant-user) 하단에 작성 및 저장된 모든 파일만 서버 재시작 시에도 영구 보존됩니다.

📌 주의사항:
홈 디렉토리 이외의 영역(예: /tmp, /etc 등)에 생성된 데이터는 서버 재시작 시 삭제될 수 있으니 반드시 주의해 주시기 바랍니다.

📌 기존 데이터:
업데이트 과정에서 파드가 재생성되므로, 현재 작업 중인 데이터는 미리 백업해 두시길 권장합니다.
