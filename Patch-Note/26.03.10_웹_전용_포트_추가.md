## [테넌트 서버 웹 전용 포트 개방 업데이트 안내]

안녕하세요. 테넌트 서버에 배포한 서비스의 외부 개방 작업을 위한 업데이트를 진행할 예정입니다. 현재 각 테넌트 서버에 접속을 위한 xxxx 포트를 개방해둔 상태에서, 외부로 서비스 개방을 위한 포트는 각 서버의 번호 순서대로 xxxx 포트로 개방할 예정입니다.

### 📌 웹 전용 포트 번호:

~~현재 각 서버의 포트번호 뒷자리를 사용하는 xxxx 포트가 해당 서버의 웹 전용 포트입니다! (예시: xxxxx번 서버 → 웹 전용 포트: xxxx번 포트)~~

⚠️ (15:40 수정) 현재 각 서버의 SSH 접속 포트번호 `+10` 을 한 포트번호가 해당 서버의 웹 전용 포트입니다! (예시: xxxx 서버 -> 웹 전용 포트: xxxx 포트)

### 📌 사용처:

현재 테넌트 서버는 쿠버네티스 환경 위에 구축한 서버이기 때문에 메모리 스왑 기능을 제공하지 않습니다. 때문에 매트릭/로그 등을 저장하는 모니터링 서버를 띄울 경우 OOM 으로 인해 서버가 종료될 수 있습니다. (가급적 무중단 배포 실습을 위한 백엔드 서버로 사용하는 것을 권장합니다!)

(**실제 테넌트에게 제공되는 포트 번호는 지웠습니다. 현재 테넌트 서버를 사용 중인 분들은 디스코드 공지 혹은 개인 연락을 통해 포트번호를 확인하시길 바랍니다.**)
